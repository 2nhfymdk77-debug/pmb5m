# 仓位控制参数说明

## 参数列表

| 参数名 | 默认值 | 说明 |
|--------|--------|------|
| 基础仓位比例 | 12 | 初始余额的1/N作为基础仓位 |
| 仓位倍增阈值 | 3 | 余额达到初始余额的N倍时，仓位翻倍 |
| 最大仓位倍数 | 8 | 仓位最大倍数限制，防止过度扩张 |
| 最小开仓金额 | 1 | 最小开仓金额（美元），低于此值等待结算 |

## 仓位计算逻辑

```
基础仓位 = 初始余额 / 基础仓位比例
实际仓位 = 基础仓位 × 仓位倍数

仓位倍数规则：
- 初始倍数 = 1
- 当余额 >= 初始余额 × 阈值 时，倍数 × 2
- 继续检查：当余额 >= 初始余额 × 阈值² 时，倍数再 × 2
- 以此类推，直到倍数达到最大值或余额不再满足条件
```

## 最小开仓金额

- 当计算出的仓位 < 最小开仓金额时，使用最小开仓金额
- 当余额 < 最小开仓金额时，不进行交易
- Polymarket 最小订单金额为 $1，建议设置为 $1-$5

## 注意事项

1. **余额不足保护**：实际仓位不会超过当前余额
2. **最小股数限制**：Polymarket 买入最少 5 股，卖出无限制
3. **仓位倍增逻辑**：只有盈利后余额增长，仓位才会倍增
4. **风险控制**：建议使用保守策略，避免过度杠杆

## GUI 配置

在 GUI 界面的"策略配置"表格中，可以直接修改以下参数：

- 第 8 行：基础仓位比例（默认 12）
- 第 9 行：仓位倍增阈值（默认 3）
- 第 10 行：最大仓位倍数（默认 8）
- 第 11 行：最小开仓金额（默认 $1）

修改参数后点击"启动交易"按钮即可生效。
