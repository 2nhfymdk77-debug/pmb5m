# 现代化界面显示文档

## 概述

Polymarket 自动交易系统 v2.0 采用现代化的终端界面设计，提供桌面应用程序级别的视觉效果。

## 主要特性

### 1. 丰富的颜色支持

#### 基础颜色
- 黑色、红色、绿色、黄色、蓝色、紫色、青色、白色
- 亮色版本 (Bright Black, Bright Red, etc.)

#### 特殊颜色
- **成功** (Success) - 绿色
- **错误** (Error) - 红色
- **警告** (Warning) - 黄色
- **信息** (Info) - 青色
- **高亮** (Highlight) - 亮黄色
- **淡化** (Muted) - 亮黑色

#### 样式
- **粗体** (Bold)
- **淡化** (Dim)
- **下划线** (Underline)
- **高亮** (Highlight)
- **闪烁** (Blink)

### 两列布局

界面采用智能的两列布局：
- **左侧列**：市场信息、当前持仓、交易统计
- **右侧列**：账户信息、挂单状态、交易参数
- **全宽**：系统状态、交易历史

## 颜色语义

### 状态颜色

| 颜色 | 用途 | 示例 |
|------|------|------|
| 绿色 | 成功、盈利、上涨 | 盈利 +$2.67、API 已连接 |

- **负盈亏**：红色 📉
- **零盈亏**：黄色 •

### 状态指示

- **已连接**：绿色 ✓
- **未连接**：红色 ✗
- **错误**：红色 ⚠
- **警告**：黄色 ⚠

## 使用示例

### 基础使用

```python
from ui_display import RealTimeDisplay

# 创建显示控制器
display = RealTimeDisplay()

# 显示完整仪表盘
display.show_full_dashboard(
    market_data={'yes_price': 75.0, 'no_price': 25.0, ...},
    balance=125.50,
    initial_balance=120.0,
    ...
)
```

### 自定义卡片

```python
from ui_display import Card, ModernFormatter

# 创建卡片
card = Card("标题", 80, 'DOUBLE')
card.add_line("内容行 1")
card.add_spacer()
card.add_line("内容行 2")

# 渲染卡片
print(card.render())
```

### 自定义进度条

```python
from ui_display import ProgressBar

# 创建进度条
progress_bar = ProgressBar()

# 渲染进度条
print(progress_bar.render(0.75))
print(progress_bar.render_with_value(75, 100))
```

## 兼容性

### 终端支持

- ✅ Linux Terminal
- ✅ macOS Terminal
- ✅ Windows Terminal (支持 ANSI 颜色)
- ✅ VS Code Integrated Terminal
- ✅ iTerm2 (macOS)

### 字体支持

推荐使用等宽字体以获得最佳效果：
- **Windows**: Consolas, Cascadia Code
- **macOS**: Monaco, Menlo
- **Linux**: Ubuntu Mono, Source Code Pro

## 注意事项

1. **终端颜色支持**：确保终端支持 ANSI 颜色代码
2. **字体选择**：使用等宽字体以获得最佳对齐效果
3. **窗口大小**：建议终端窗口宽度至少 120 列
4. **性能优化**：频繁更新时，考虑减少刷新频率

## 技术实现

### ANSI 颜色代码

```python
# 基础颜色
BLACK = '\033[30m'
RED = '\033[31m'
GREEN = '\033[32m'
YELLOW = '\033[33m'
BLUE = '\033[34m'
MAGENTA = '\033[35m'
CYAN = '\033[36m'
WHITE = '\033[37m'

# 样式
BOLD = '\033[1m'
DIM = '\033[2m'
UNDERLINE = '\033[4m'

# 重置
RESET = '\033[0m'
```

### Unicode 字符

使用 Unicode 字符绘制边框和图标：
- 边框：┌ ┐ └ ┘ ─ │ ┼ 等
- 图标：✓ ✗ ⚠ ℹ 📈 📉 等
