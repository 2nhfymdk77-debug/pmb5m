# Polymarket Auto Trader

一个用于 Polymarket 平台 5 分钟比特币预测的自动化交易程序，采用极简界面设计，支持模拟测试和手动参数配置。

## 功能特性

### 核心策略
- ✅ **自动仓位管理**：每次使用账户余额的 1/12 作为仓位
- ✅ **双向挂单**：在指定价格（默认 $75）同时挂多单和空单
- ✅ **智能止损止盈**：成交后自动设置 45 止损、95 止盈
- ✅ **定时交易**：每 5 分钟执行一次交易周期
- ✅ **杠杆倍增**：当余额达到初始余额的 3 倍时，仓位自动翻倍

### 交易功能
- ✅ **模拟测试模式**：无需真实资金即可测试策略
- ✅ **手动参数配置**：支持自定义开仓价、止损价、止盈价
- ✅ **交易统计**：实时显示总交易次数、盈利次数、亏损次数、总盈亏
- ✅ **交易历史**：完整记录每一笔订单的详细信息

### 界面特性
- ✅ **极简设计**：清晰的界面布局，操作简单直观
- ✅ **实时数据**：显示当前价格、账户余额、杠杆倍数
- ✅ **状态监控**：底部状态栏显示运行状态和上次交易时间

## 本地开发环境搭建

### 前置要求

在开始之前，请确保你的本地环境已安装：

- **Node.js**：版本 18.x 或更高（推荐 20.x）
- **pnpm**：推荐的包管理器

```bash
# 检查 Node.js 版本
node --version

# 安装 pnpm（如果尚未安装）
npm install -g pnpm
```

### 开发步骤

1. **克隆或下载项目代码**

   将项目文件下载到本地目录，例如：`C:\polymarket-auto-trader\`

2. **安装依赖**

   在项目根目录下打开终端/命令提示符，执行：

   ```bash
   cd C:\polymarket-auto-trader
   pnpm install
   ```

3. **启动开发模式**

   ```bash
   pnpm start
   ```

   这将启动 Electron 应用，你可以在开发模式下测试所有功能。

## 打包成 Windows 桌面应用

### 方法一：使用 pnpm 打包（推荐）

1. **确保依赖已安装**

   ```bash
   pnpm install
   ```

2. **执行打包命令**

   ```bash
   pnpm run dist
   ```

3. **获取安装包**

   打包完成后，安装包位于 `dist/` 目录下，文件名为：
   ```
   Polymarket Auto Trader Setup 1.0.0.exe
   ```

   双击运行即可安装到你的 Windows 系统。

### 方法二：使用 npm 打包

如果你更喜欢使用 npm：

1. **安装依赖**

   ```bash
   npm install
   ```

2. **执行打包**

   ```bash
   npm run dist
   ```

### 打包配置说明

打包配置已在 `package.json` 中预设：

```json
{
  "build": {
    "appId": "com.polymarket.autotrader",
    "productName": "Polymarket Auto Trader",
    "win": {
      "target": "nsis"
    }
  }
}
```

- **target: nsis**：生成 Windows 安装程序
- **appId**：应用唯一标识
- **productName**：应用显示名称

## 使用说明

### 首次使用

1. **启动应用**

   - 开发模式：运行 `pnpm start`
   - 打包后：双击桌面图标或从开始菜单启动

2. **配置交易参数**

   在界面右侧的"交易参数"面板中设置：

   - **开仓价位**：默认 $75，设置同时挂多空单的价格
   - **仓位比例**：默认 12，表示每次使用余额的 1/12
   - **止损价位**：默认 $45，触发自动平仓的价格
   - **止盈价位**：默认 $95，触发自动平仓的价格
   - **初始余额**：默认 $1000，用于计算杠杆倍数
   - **交易周期**：默认 5 分钟，自动交易的间隔

3. **选择运行模式**

   - **模拟测试**：点击"模拟测试"按钮，使用模拟数据测试策略
   - **真实交易**：点击"开始交易"按钮（需要连接 Polymarket API）

### 模拟测试模式

1. 点击"模拟测试"按钮
2. 应用会自动：
   - 重置模拟余额为初始余额
   - 每 30 秒执行一次交易周期（模拟 5 分钟）
   - 随机生成价格波动
   - 记录所有交易结果

3. 观察交易统计和历史记录
4. 测试完成后点击"停止交易"

### 真实交易模式

⚠️ **注意**：当前版本为模拟版本，真实交易需要集成 Polymarket API。

### 交易统计功能

应用底部显示实时统计：

- **总交易次数**：所有已完成的交易
- **盈利次数**：盈利的交易笔数
- **亏损次数**：亏损的交易笔数
- **总盈亏**：累计盈亏金额（绿色为盈利，红色为亏损）

### 交易历史记录

在"交易历史"面板中可以查看：

- **时间**：交易执行时间
- **方向**：做多/做空
- **开仓价**：订单执行价格
- **平仓价**：订单平仓价格
- **仓位**：交易金额
- **盈亏**：该笔交易的盈亏
- **状态**：已完成/待成交

支持按类型筛选：
- **全部**：显示所有记录
- **盈利**：仅显示盈利交易
- **亏损**：仅显示亏损交易
- **待成交**：仅显示挂单

## 交易策略详解

### 仓位管理

1. **初始阶段**：每次使用余额的 1/12
2. **杠杆阶段**：当余额 ≥ 初始余额 × 3 时，杠杆倍数 = floor(当前余额 / 初始余额)
   - 例如：初始 $1000，当前 $3500，杠杆 = 3x
   - 仓位 = 余额 / 12 × 杠杆倍数

### 止损止盈机制

- **做多（Long）**：
  - 价格 ≤ 止损价：触发止损
  - 价格 ≥ 止盈价：触发止盈

- **做空（Short）**：
  - 价格 ≥ 止损价：触发止损
  - 价格 ≤ 止盈价：触发止盈

### 交易周期

- 按设定的交易周期（默认 5 分钟）自动执行
- 每个周期执行流程：
  1. 读取当前余额
  2. 计算杠杆倍数
  3. 计算仓位大小
  4. 在开仓价同时挂多空单
  5. 一侧成交后取消另一侧
  6. 设置止损止盈
  7. 等待平仓或触发止损止盈

## 注意事项

### 风险提示

- ⚠️ **自动交易涉及金融风险**，请谨慎使用

- ⚠️ **模拟结果不代表真实市场表现**
- ⚠️ **请合理设置止损止盈**，控制风险

### 数据安全

- 所有配置和交易历史保存在本地
- 不会上传任何数据到远程服务器

当前版本为**纯模拟版本**，不包含真实 API 集成。

如需连接 Polymarket 真实交易，需要：

1. 获取 Polymarket API 密钥
2. 在代码中集成 API 调用
3. 实现真实的订单下单和查询功能

## 常见问题

### Q1: 打包时出现错误怎么办？

A: 确保 Node.js 版本 ≥ 18.x，并且已安装所有依赖：

```bash
node --version
pnpm install
```

### Q2: 如何重置交易历史？

A: 删除配置目录下的 `trade-history.json` 文件，或在应用中添加清空功能。

### Q3: 模拟测试和真实交易的区别？

A:
- **模拟测试**：使用随机价格波动，不消耗真实资金
- **真实交易**：需要连接 Polymarket API，进行真实资金交易

### Q4: 如何修改默认参数？

A: 在界面右侧的"交易参数"面板中修改，或直接编辑 `config.json` 文件。

### Q5: 应用无法启动怎么办？

A: 检查以下几点：
1. Node.js 版本是否正确
2. 是否已安装所有依赖
3. 查看 console.log 输出（开发模式按 F12）
