# Polymarket 自动交易系统 - V4策略自动启动版

## 功能特点

- ✅ **自动启动**：程序运行后直接开始交易，无需任何配置
- ✅ **V4策略**：使用CONTINUOUS模式，连续交易策略
- ✅ **实时监控**：实时显示YES/NO价格、剩余时间、余额、盈亏
- ✅ **交易历史**：记录所有交易详情
- ✅ **运行日志**：实时显示交易日志

## V4策略参数（固定）

| 参数 | 值 | 说明 |
|------|-----|------|
| 策略模式 | CONTINUOUS | 连续交易模式 |
| 买入价格 | 70% | 价格达到70%时买入 |
| 止损价格 | 45% | 价格跌到45%时止损 |
| 止盈价格 | 95% | 价格涨到95%时止盈 |
| 买入限制 | 85% | 价格超过85%不买入 |
| 基础仓位 | 1/12 | 初始余额的1/12 |
| 仓位倍增阈值 | 3倍 | 余额达到3倍时翻倍 |
| 最大仓位倍数 | 8倍 | 防止过度扩张 |
| 最小开仓金额 | $1 | 最小开仓金额 |

## 运行方式

### 方式1：直接运行Python脚本

```bash
python app_gui.py
```

### 方式2：打包为EXE后运行

```bash
# 安装打包工具
pip install pyinstaller

# 打包
pyinstaller --onefile --windowed --name "PolymarketTrader" app_gui.py

# 运行生成的EXE文件
dist/PolymarketTrader.exe
```

## 使用流程

1. **启动程序**：双击运行 `app_gui.py` 或打包后的EXE文件
2. **自动交易**：程序自动连接API并开始交易
3. **监控状态**：实时查看交易状态、价格、余额等信息
4. **停止交易**：点击"停止交易"按钮停止运行
5. **退出程序**：关闭窗口即可退出

## 前置配置

在运行前需要配置API密钥：

1. 创建 `.env` 文件
2. 填写以下信息：

```env
PRIVATE_KEY=你的私钥
API_KEY=你的API密钥
API_SECRET=你的API密钥密文
PASSPHRASE=你的API密码
```

## 界面说明

### 实时状态区域
- **状态**：当前交易状态（空闲/持仓/监控中）
- **YES价格**：YES代币当前价格
- **NO价格**：NO代币当前价格
- **剩余时间**：当前事件剩余时间
- **余额**：账户当前余额
- **总盈亏**：累计盈亏金额

### 交易历史区域
- 记录所有交易详情
- 包含时间、代币、方向、价格、盈亏等信息
- 自动滚动显示最新记录

### 运行日志区域
- 实时显示交易日志
- 包含启动信息、交易记录、错误信息等
- 黑色背景，易于阅读

### 控制按钮
- **停止交易**：点击后停止交易循环
- 关闭窗口会自动停止交易并退出

## 注意事项

⚠️ **重要提示**：
1. 确保API密钥配置正确
2. 确保账户有足够余额
3. 程序启动后立即开始交易，无需确认
4. 关闭窗口前会提示确认（如果正在交易）

## 策略说明

### V4策略特点
- **连续交易**：无持仓时立即寻找下一个交易机会
- **快速响应**：价格达到目标立即执行
- **风险控制**：固定止损45%、止盈95%
- **仓位管理**：根据余额动态调整仓位

### 交易流程
1. 监控市场价格
2. 当价格达到买入条件（≤70%）时买入
3. 设置止损（45%）和止盈（95%）
4. 监控价格触发止损或止盈
5. 卖出后计算盈亏
6. 继续下一轮交易
